# Moonschool - English Language School Website

A modern, responsive landing page for an English Language School.
Designed with a focus on conversion, clarity, and professional aesthetics.

## Features
- **Modern UI**: Clean design inspired by top EdTech platforms (Skillbox, etc.).
- **Bilingual**: Fully functional English/Russian language toggle.
- **Responsive**: Adapts perfectly to mobile, tablet, and desktop.
- **Lead Generation**: Integrated contact form (demo) and Call-to-Action buttons.

## Project Structure
- `index.html`: Main landing page.
- `style.css`: All styles and responsive rules.
- `script.js`: Logic for language toggle and form handling.

## How to Run Locally
Simply open `index.html` in any web browser. No server required.

## Deployment (GitHub Pages)
1. Initialize a git repository: `git init`
2. Add files: `git add .`
3. Commit: `git commit -m "Initial commit"`
4. Push to your GitHub repository.
5. Go to Repository Settings -> Pages -> Select 'main' branch -> Save.
